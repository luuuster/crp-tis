CRP DS · Button

1. Matriz — Intent × Estilo

5 intents (Primária, Secundária, Danger, Warning, Success) × 5 estilos (Sólido, Outline, Soft, Ghost, Link). Tamanho M.

2. Tamanhos — Pequeno · Normal · Grande

Padding/fonte/gap vêm de --space-* e --font-size-*. Cada estilo nos 3 tamanhos (intent Primária).

3. Estados — Normal · Hover · Active · Focus · Desabilitado · Loading

Hover/Active/Focus aqui são forçados via classe para você ver todos de uma vez. Foco usa --ring; desabilitado usa --opacity-disabled; loading usa --animate/--duration.

4. Ícones — esquerda · direita · ambos · só ícone

Gap entre ícone e texto via --space-*. "Só ícone" vira quadrado (aspect-ratio:1).

5. Tudo junto — intents × estados (Sólido, M, com ícone)

Visão de stress: cada intent em cada estado.
